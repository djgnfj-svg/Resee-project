# 주간 시험

## 핵심 개념

학습한 콘텐츠를 기반으로 **AI가 자동으로 문제를 생성**하여 주간 단위로 실력을 점검합니다.
객관식과 O/X 문제로 구성되며, 즉시 피드백을 제공합니다.

## 주요 기능

- **AI 문제 생성** (Claude 3 Haiku):
  - 객관식 (4지선다)
  - O/X (참/거짓)
- **자동 채점**: 정답 즉시 확인
- **상세 해설**: 문제별 설명 제공
- **결과 저장**: 시험 이력 관리

## 동작 흐름

```
1. 사용자가 주간 시험 시작
   ↓
2. 학습한 콘텐츠 기반으로 AI가 문제 생성
   ↓
3. 문제 풀이 (객관식/O/X)
   ↓
4. 답안 제출 → 자동 채점
   ↓
5. 점수 + 해설 제공
   ↓
6. 결과 저장
```

## API

- `GET /api/weekly-test/` - 시험 목록
- `POST /api/weekly-test/` - 시험 생성 (AI 문제 생성)
- `GET /api/weekly-test/{id}/` - 시험 상세
- `POST /api/weekly-test/start/` - 시험 시작
- `POST /api/weekly-test/submit-answer/` - 답안 제출
- `POST /api/weekly-test/complete/` - 시험 완료
- `GET /api/weekly-test/{id}/results/` - 결과 조회

## 관련 파일

- `backend/weekly_test/views.py` - 시험 API
- `backend/weekly_test/ai_service.py` - AI 문제 생성
- `backend/weekly_test/models.py` - WeeklyTest, Question, Answer
- `frontend/src/pages/WeeklyTestPage.tsx` - 시험 UI
