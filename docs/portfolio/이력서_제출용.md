### 주요 기능 및 성과

#### 1. Ebbinghaus 망각곡선 기반 복습 스케줄링 자동화
- Hermann Ebbinghaus 이론 기반 8단계 복습 간격(1, 3, 7, 14, 30, 60, 120, 180일) 자동 계산
- 구독 티어별 차등 제공(FREE: 2단계, BASIC: 7단계, PRO: 8단계)
- Django Signal로 콘텐츠 생성 시 ReviewSchedule 자동 생성
- **→ [상세 보기](블로그URL)**

#### 2. AI 기반 서술형 답변 자동 평가 (Claude API)
- Anthropic Claude API로 사용자 답변 자동 채점(0-100점)
- 무의미한 답변 필터링(숫자/스팸 → 0점), 70점 이상 자동 "remembered" 판정
- 한국어 피드백 생성으로 복습 품질 향상
- **→ [상세 보기](블로그URL)**

#### 3. Redis 캐싱으로 응답 속도 80% 개선
- Cache-aside Pattern 적용, Hit Rate 85%+ 달성
- API 응답: 250ms → 50ms, DB 부하 80% 감소
- 동시 접속 1,000명 처리 가능
- **→ [상세 보기](블로그URL)**

#### 4. N+1 쿼리 301개 → 3개로 99% 감소
- select_related(), prefetch_related() 최적화
- 복합 인덱스 3개 설계로 조회 성능 10배 향상
- 페이지 로딩: 500ms → 50ms (90% 개선)
- **→ [상세 보기](블로그URL)**

#### 5. Celery + Redis 비동기 작업 큐 구축
- 대용량 이메일/백업 작업 비동기 처리
- pg_dump 자동 백업, 3회 재시도, Slack 실시간 알림
- 데이터 유실 위험 제로
- **→ [상세 보기](블로그URL)**

#### 6. JWT 인증 + Rate Limiting 보안 강화
- 이메일 토큰 SHA-256 해싱, Timing Attack 방어
