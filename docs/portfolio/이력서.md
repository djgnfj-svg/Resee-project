## 주요 기능 및 성과

### 1. Ebbinghaus 망각곡선 기반 복습 스케줄링 자동화 ⭐⭐⭐⭐⭐
Hermann Ebbinghaus 이론 기반 최적 복습 간격(1, 3, 7, 14, 30, 60, 120, 180일) 자동 계산, 구독 티어별 차등 제공(FREE: 2단계, BASIC: 7단계, PRO: 8단계), Django Signal로 콘텐츠 생성 시 ReviewSchedule 자동 생성
**→ [복습 스케줄링 보기](https://yourblog.com/resee/복습스케줄링)**

### 2. AI 기반 서술형 답변 자동 평가 (Claude API) ⭐⭐⭐⭐⭐
Anthropic Claude API로 사용자 답변 자동 채점(0-100점), 무의미한 답변 필터링(숫자/스팸 → 0점), 70점 이상 자동 "remembered" 판정, 한국어 피드백 생성으로 복습 품질 향상
**→ [AI 답변 평가 보기](https://yourblog.com/resee/AI평가)**

### 3. Redis 캐싱 전략 (Cache-aside Pattern) ⭐⭐⭐⭐⭐
Cache-aside Pattern으로 자주 조회되는 복습 목록·통계 데이터 캐싱, Hit Rate **85%+** 달성으로 API 응답 **250ms → 50ms (80% 단축)**, DB 부하 **80% 감소**
**→ [캐싱 전략 보기](https://yourblog.com/resee/11)**

### 4. GitHub Actions CI/CD + AWS EC2 배포 자동화 ⭐⭐⭐⭐⭐
deploy.sh 380줄 bash script로 Docker 자동 설치, Swap 메모리 추가, 환경변수 검증, Health check 대기 구현, 배포 시간 **30분 → 5분 (83% 단축)**
**→ [배포 파이프라인 보기](https://yourblog.com/resee/01)**

### 5. Celery + Redis 비동기 작업 처리 ⭐⭐⭐⭐⭐
pg_dump+gzip 자동 백업(매일 3시), 3회 재시도, 10분 timeout, Slack 성공/실패 알림으로 데이터 유실 위험 **제로**
**→ [비동기 처리 보기](https://yourblog.com/resee/02)**

### 6. JWT + OAuth 2.0 + 보안 강화 ⭐⭐⭐⭐⭐
이메일 토큰 SHA-256 해싱, `secrets.compare_digest()` Constant-time 비교로 DB 유출·Timing Attack 방어
**→ [보안 강화 보기](https://yourblog.com/resee/03)**

## 트러블슈팅 경험

*더 많은 내용은 블로그를 참고해주세요*

### 1. N+1 쿼리 최적화 (**301개 → 3개, 99% 감소**) ⭐⭐⭐⭐⭐
복습 목록 조회 시 301개 쿼리 발생 문제를 `select_related()`와 복합 인덱스 3개 설계로 3개로 감소, 응답 시간 **500ms → 50ms (90% 개선)**
**→ [최적화 과정 보기](https://yourblog.com/resee/05)**

### 2. Redis 기반 Rate Limiting 마이그레이션 (정확도 **50% → 100%**) ⭐⭐⭐⭐⭐
locmem 캐시의 Worker별 독립 캐싱 문제를 Redis 중앙 집중식 캐시로 전환, 멀티 프로세스 환경에서 Rate Limit 정확도 **2배 향상**, Scale-out 준비 완료
**→ [마이그레이션 과정 보기](https://yourblog.com/resee/10)**

### 3. 이메일 토큰 보안 강화 (평문 → SHA-256 해싱) ⭐⭐⭐⭐
DB 유출 시나리오 대응을 위해 평문 토큰을 SHA-256 해싱으로 전환, `secrets.compare_digest()`로 Timing Attack 방어
**→ [보안 강화 과정 보기](https://yourblog.com/resee/06)**

### 4. Celery 시작 순서 문제 (초기화 에러 **0건**) ⭐⭐⭐⭐
Docker Compose `depends_on` + Health check로 PostgreSQL → Backend → Celery 순서 보장, 컨테이너 초기화 에러 완전 제거
**→ [문제 해결 과정 보기](https://yourblog.com/resee/07)**

## 💡 2025년 백엔드 신입 필수 역량

### **핵심 구현 6가지**
1. ⭐⭐⭐⭐⭐ **Ebbinghaus 복습 스케줄링 자동화** (도메인 핵심 로직, Signal 자동화)
2. ⭐⭐⭐⭐⭐ **AI 답변 자동 평가** (Claude API 통합, 0-100점 채점)
3. ⭐⭐⭐⭐⭐ **Redis 캐싱 전략** (API 응답 80% 단축, 2025년 핵심 최적화 도구)
4. ⭐⭐⭐⭐⭐ **GitHub Actions CI/CD + AWS 배포** (83% 단축, 2025년 표준)
5. ⭐⭐⭐⭐⭐ **Celery + Redis 비동기 작업 처리** (데이터 유실 제로)
6. ⭐⭐⭐⭐⭐ **JWT + OAuth 2.0 보안 강화** (Timing Attack 방어)

### **트러블슈팅 역량 4가지**
- ⭐⭐⭐⭐⭐ **N+1 쿼리 최적화** (301→3개, 99% 감소, 면접 필수 질문)
- ⭐⭐⭐⭐⭐ **Redis Rate Limiting 마이그레이션** (멀티 프로세스 환경 대응)
- ⭐⭐⭐⭐ **이메일 토큰 보안 강화** (SHA-256, Timing Attack 방어)
- ⭐⭐⭐⭐ **Celery 시작 순서** (Docker 의존성 관리)
