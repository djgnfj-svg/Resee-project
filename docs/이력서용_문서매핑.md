2. select_related() vs prefetch_related() 차이
3. Mixin 패턴으로 코드 재사용
4. 복합 인덱스 설계 (user + next_review_date + is_active)
5. 측정 가능한 성과 (99% 감소, 90% 응답 속도 개선)
