# 🎓 Resee 사용자 가이드

## 📖 개요

**Resee**는 에빙하우스 망각곡선 이론을 기반으로 한 스마트 복습 플랫폼입니다. 
학습한 내용을 과학적으로 검증된 간격으로 복습하여 장기기억으로 전환하는 것을 도와줍니다.

### 🎯 핵심 기능
- **스마트 복습 스케줄링**: 에빙하우스 망각곡선 기반 최적 복습 타이밍
- **AI 문제 생성**: Claude AI를 활용한 맞춤형 문제 자동 생성  
- **학습 분석**: 개인 학습 패턴 분석 및 성과 추적
- **구독 기반 서비스**: 단계별 기능 제공 (FREE/BASIC/PRO)

---

## 🚀 빠른 시작 가이드

### 1. 회원가입 및 로그인
1. **이메일 회원가입**: 이메일 인증 후 계정 활성화
2. **Google 소셜 로그인**: 원클릭 간편 가입/로그인
3. **이메일 인증**: 구독 기능 사용을 위해 이메일 인증 필수

### 2. 첫 학습 콘텐츠 생성
1. **콘텐츠 추가** 버튼 클릭
2. **제목, 내용, 카테고리** 입력
3. **TipTap 에디터** 활용하여 풍부한 텍스트 작성
4. 저장 시 **자동으로 복습 스케줄** 생성

### 3. 복습하기
1. **오늘의 복습** 섹션에서 복습할 콘텐츠 확인
2. 콘텐츠 내용 숙지 후 **기억 정도 선택**:
   - 🟢 **완전히 기억함**: 다음 간격으로 진행
   - 🟡 **부분적으로 기억**: 같은 간격 반복
   - 🔴 **기억하지 못함**: 1일 후 다시 복습

## 🔄 복습 시스템 상세

### 에빙하우스 망각곡선 간격
각 구독 단계별로 지원하는 복습 간격이 다릅니다:

#### 📦 FREE 구독 (무료)
- **복습 간격**: 1일, 3일 (최대 3일)
- **AI 문제**: 지원 안함
- **API 요청**: 500회/시간

#### 💎 BASIC 구독  
- **복습 간격**: 1일, 3일, 7일, 14일, 30일, 60일, 90일 (최대 90일)
- **AI 문제**: 30문제/일
- **API 요청**: 1000회/시간

#### 🚀 PRO 구독
- **복습 간격**: 1일, 3일, 7일, 14일, 30일, 60일, 120일, 180일 (최대 180일)
- **AI 문제**: 200문제/일 (거의 무제한)
- **API 요청**: 2000회/시간

### 복습 결과에 따른 스케줄 조정
- **잘 기억함**: 다음 단계 간격으로 진행 (1일 → 3일 → 7일...)
- **부분 기억**: 현재 간격 유지 (3일 → 3일)
- **기억 못함**: 첫 단계로 리셋 (7일 → 1일)

## 🤖 AI 문제 생성 기능

### 지원 문제 유형
1. **객관식 문제**: 4지선다형 문제

2. **구독 단계별 일일 한도** 확인
3. Claude AI가 콘텐츠 분석하여 **맞춤형 문제 생성**
4. **즉시 문제 풀이** 또는 나중에 풀이 가능

### 문제 풀이 및 피드백
- **정답/오답 즉시 확인**
- **상세한 해설** 제공
- **학습 성과 추적** 및 분석

## 🛠️ 문제 해결

### 자주 묻는 질문 (FAQ)

#### Q: 복습을 놓쳤어요. 어떻게 되나요?
**A**: 놓친 복습은 "오늘의 복습"에 계속 표시되며, 언제든 진행 가능합니다. 
복습 완료 시 원래 스케줄대로 다음 복습이 예약됩니다.

#### Q: AI 문제가 생성되지 않아요.
**A**: 다음을 확인해주세요:
- 구독 상태 (FREE는 AI 기능 미지원)
- 일일 한도 소진 여부
- 콘텐츠 길이 (너무 짧으면 문제 생성 어려움)

#### Q: 이메일 인증을 받지 못했어요.
**A**: 스팸 폴더를 확인하거나, 설정에서 이메일 재발송을 요청하세요.

- **브라우저 캐시 정리**: 가끔 브라우저 캐시를 정리하면 성능 향상

## 🎯 학습 효과 극대화 팁

3. **일관성 유지**: 매일 일정한 시간에 복습하기
4. **환경 조성**: 집중할 수 있는 조용한 환경

- **취약 카테고리** 파악하여 집중 학습
